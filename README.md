# Role-Based Authentication System (Admin + User)

A full-stack role-based authentication system built using:

- **Backend:** Node.js, Express, Prisma, PostgreSQL (Neon DB)
- **Frontend:** Next.js 14 (App Router), Tailwind CSS
- **Auth:** JWT Token + Middleware Protection

This project demonstrates clean and secure role-based access:
- Regular **User** can access general features.
- **Admin** can access advanced features like managing users & viewing sensitive data.

Perfect for interviews, assignments, and production-level learning.
